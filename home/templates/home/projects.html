Spotify Custom Shuffle

I started this project due to my own dissatisfaction with Spotify's shuffle algorithm. In its current state, this is a simple Python script that re-orders the songs in the user's public playlist according to a set of parameters. It uses an algorithm that roughly distributes songs by an artist evenly throughout the playlist. For example, if a playlist contains 25 songs, 5 of which are by Pink Floyd, then for each "chunk" of 5 songs, 1 will be by Pink Floyd. The user can set chunk width, or number of chunks. Check it out on GitHub. Technologies used:

  • Python
  • Spotipy (Python library for the Spotify Web API)
